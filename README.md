### Summary

SvelteKit + express API, chatGPT code sample

***
### Setup

```
npm install
```

***
* server/.env : postgres set

```
POSTGRES_DATABASE="chat"
POSTGRES_USER=""
POSTGRES_PASSWORD=""
POSTGRES_HOST=""
POSTGRES_PORT=5432

```

***
### Start server
* start :

```
# server
yarn dev

# client
yarn dev

```
***
### build

```
#server 

cd server

yarn build

yarn start

```
***

* open: http://localhost:5173/
